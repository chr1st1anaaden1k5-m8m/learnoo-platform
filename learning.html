Introduction to Database Systems

📌 What is a Database?

A database is an organized collection of data that is stored and accessed electronically from a computer system.

Why Do We Need Databases?
  • ✅ To store large amounts of data efficiently
  • ✅ Fast searching and retrieval
  • ✅ Multiple users can access data at the same time
  • ✅ Data security and backup
Types of Databases

1. Relational Databases (SQL) — MySQL, PostgreSQL, SQLite
2. NoSQL Databases — MongoDB, Firebase, Cassandra
